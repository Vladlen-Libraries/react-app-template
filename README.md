#  React SPA GetGain Template

## Install

```sh
npx create-react-app my-app --template https://github.com/Vladlen-Libraries/react-app-template.git
```

### Includes
- Module architecture
- TypeScript
- Domain JS
- Redux Toolkit
- Permissions
- UD-Form
- UD-UI
- Auth by phone

### Not Includes
- SSR
- Theming
- Lists
